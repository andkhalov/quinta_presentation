# Инструкции для Claude Code

## Перезапуск сайта

**ВАЖНО:** После любых изменений в файлах проекта (шаблоны, CSS, JS, Python-код) необходимо перезапускать сайт целиком, иначе изменения не будут видны.

Команда перезапуска:
```bash
cd /home/debian/quinta_presentation && ./run-all.sh
```

Django dev-сервер в этом проекте НЕ отслеживает изменения автоматически. Всегда перезапускай сайт после внесения правок.
