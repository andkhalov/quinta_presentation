# Agent.md - Руководство по созданию комплекса материалов для Quinta Essentia

## Цель проекта
Создать комплексный набор маркетинговых материалов для программы сетевого маркетинга Quinta Essentia, включающий:
1. Мета-анализ выступлений спикеров
2. Интерактивное веб-приложение с математической моделью маркетинга
3. Презентацию с текстом для спикера

## Исходные материалы
- `raw_materials/IMG_6008.txt` - расшифровка выступления Бориса Кшталта (PM International)
- `raw_materials/Презентация Варлам и Нина.txt` - расшифровка выступления семейной пары (PM International)
- `raw_materials/ТЗ маркетинга Quinta Essentia v.7.docx` - техническое задание маркетинговой программы
- `raw_materials/3 итерация.pptx` - текущая презентация

## План действий

### Этап 1: Мета-анализ выступлений
**Файл:** `meta_analysis.md`

1. Выявить общую структуру обоих выступлений
2. Определить ключевые этапы воздействия на аудиторию
3. Выделить эмоциональные триггеры и иррациональные ходы
4. Сформулировать универсальную структуру презентации

**Структура анализа:**
- Вступление (установление доверия)
- Представление компании
- Демонстрация результатов
- Объяснение продукта
- Разбор маркетинг-плана (деньги!)
- Истории успеха
- Призыв к действию

### Этап 2: Интерактивное приложение
**Директория:** `quinta_app/`

**Технологии:**
- Python + Django (легкий, без тяжелых фреймворков)
- HTML/CSS/JavaScript для интерактивности
- Chart.js для графиков и диаграмм
- Docker/Docker Compose для развертывания

**Функциональность:**
1. **Калькулятор доходов** - интерактивный расчет потенциального заработка
2. **Визуализация рангов** - наглядная карьерная лестница
3. **Симулятор структуры** - моделирование команды
4. **Справочник по бонусам** - все виды вознаграждений с примерами
5. **Игровые сценарии** - различные стратегии работы
6. **Навигация по ТЗ** - удобный интерфейс для изучения программы

**Ключевые разделы:**
- Кэшбэк (5% - 12.5%)
- Бонус продаж (2.5% - 7.5%)
- Командный бонус (до 7 уровней)
- Ранговый бонус
- Лидерский бонус
- ТОП бонус

### Этап 3: Презентация
**Файлы:** 
- `presentation_content.docx` - контент презентации
- `speaker_notes.md` - заметки для спикера

**Структура презентации:**

1. **Слайд 1: Хук** 
   - "Сколько вы хотите зарабатывать?"
   - Эмоциональный триггер

2. **Слайд 2: Доверие к компании**
   - Quinta Essentia - надежность
   - Продукция quinta.pro

3. **Слайд 3: Сколько можно заработать**
   - Конкретные цифры
   - Примеры реальных доходов

4. **Слайд 4: Как это работает**
   - Простая схема: клиенты + команда

5. **Слайд 5: Роли в системе**
   - Клиент vs Консультант
   - Путь развития

6. **Слайд 6: Кэшбэк - первые деньги**
   - 5% → 7.5% → 10% → 12.5%
   - Личный товарооборот

7. **Слайд 7: Бонус продаж**
   - Доход от команды
   - Таблица процентов

8. **Слайд 8: Командный бонус**
   - До 7 уровней глубины
   - Пассивный доход

9. **Слайд 9: Карьерная лестница**
   - 12 рангов от Novus до Principal
   - Визуализация пути

10. **Слайд 10: Ранговые подарки**
    - От 100 PV до недвижимости
    - Путешествия, часы, авто

11. **Слайд 11: Стратегии работы**
    - Для интровертов
    - Для активных продавцов
    - Для лидеров команд

12. **Слайд 12: Подходит всем**
    - Истории успеха
    - Разные типажи людей

13. **Слайд 13: Программа обучения**
    - Поддержка команды
    - Готовые инструменты

14. **Слайд 14: Призыв к действию**
    - Минимальный вход: 35 PV
    - Первый шаг сегодня

## Эмоциональные триггеры (из анализа выступлений)

1. **История "из грязи в князи"** - личная история трансформации
2. **Конкретные цифры** - "тысячу долларов в день"
3. **Социальное доказательство** - "тридцать тысяч человек зарабатывают"
4. **Срочность** - "впервые за 25 лет"
5. **Простота** - "один стакан утром, один вечером"
6. **Авторитет** - "миллиардная компания", "профессиональные спортсмены"
7. **Визуализация успеха** - путешествия, авто, признание
8. **Призыв к немедленному действию** - "прямо сейчас"

## Стратегии работы (игровые сценарии)

### 1. "Клиентский путь" (Интроверт)
- Фокус на личном потреблении
- Кэшбэк 5-12.5%
- Постепенное привлечение знакомых
- Цель: стабильный пассивный доход

### 2. "Продавец" (Экстраверт)
- Активные продажи клиентам
- Максимальный кэшбэк 12.5%
- Бонус продаж до 7.5%
- Цель: высокий активный доход

### 3. "Строитель команды" (Лидер)
- Фокус на привлечении консультантов
- Командный бонус с 7 уровней
- Лидерский бонус 2.5% с Doctus+
- Цель: масштабируемый пассивный доход

### 4. "Гибридная стратегия" (Универсал)
- Баланс продаж и рекрутинга
- Все виды бонусов
- Диверсификация дохода
- Цель: максимальный потенциал
